# Development Demo Script

This is a draft 60-90 second demo outline for the Alpha.

## Flow

1. Launch Sumi and show a normal webpage loading in a real browser tab.
2. Show the sidebar with tabs, spaces, and profiles.
3. Switch spaces to show workspace-oriented organization.
4. Open Glance over the current tab.
5. Close Glance quickly, then open another Glance and expand it into a normal
   tab.
6. Move a Glance page into split view.
7. Show split view with multiple views, up to four if the demo content fits.
8. Show essentials and pinned items in the sidebar.
9. Demonstrate launcher behavior conceptually: visible identity remains while
   inactive live runtime state can be unloaded.
10. Use the floating bar for address/search, suggestions, and filling an active
    split slot.
11. Show bookmarks, history, and settings search briefly.
12. Show memory modes and inactive tab unloading settings.
13. If safe, show the extension UI and Safari compatibility report.
14. End with the remaining Alpha hardening work: Safari password-manager
    support and published update validation.

## Do Not Show

- Personal tabs.
- Login sessions.
- API keys.
- Private repositories.
- Local paths unless necessary.
- Unannounced secrets or credentials.
- Password-manager accounts or vault contents.
- Unsupported benchmark claims.

File Picker

Browser UI Expectations

  • File picker is one-time and native picker mediated.
  • No saved Allow/Block option is offered for file picker.
  • No stored site permission is written when a picker opens, returns files, or is cancelled.
  • Privacy Site Settings does not list file picker as a saved exception; it may show explanatory recent activity only.
  • Selected file names and counts can be shown; full local paths must never be exposed.

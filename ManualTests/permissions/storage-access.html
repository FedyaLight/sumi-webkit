Storage Access Overview

Browser UI Expectations

  • Storage Access API is meaningful for an embedded third-party frame, not a same-origin top-level page.
  • First-time third-party document.requestStorageAccess() should show the URL-bar authorization prompt in an active normal tab when WebKit exposes Sumi's callback.
  • Allow and Don't allow are saved decisions for normal profiles; dismiss denies/cancels the current request without persistence.
  • URL hub -> Permissions and Privacy -> Site Settings show Embedded content access / Storage Access decisions.

Two-Origin / Two-Port Setup

Run two local servers from this directory. This two-origin, two-port setup uses the top-level embedder from port 8000 and the requesting frame from port 8001.

python3 -m http.server 8000
python3 -m http.server 8001

External App Links

Browser UI Expectations

  • First-time user-click external app links show the URL-bar authorization prompt.
  • Open this time opens only the current attempt. Always allow is site and scheme specific.
  • Background/script attempts block by default unless an allow exists.
  • No external app should open before the user settles a prompt or a stored allow is found.
  • URL query and fragment are redacted in stored attempt state.

Pop-ups And Redirects

Browser UI Expectations

  • User-activated popups open by default unless the site is blocked.
  • Background/script popups block by default unless the site is allowed.
  • Blocked state appears in the URL-bar indicator and URL hub -> Permissions current-page status.
  • URL hub -> Permissions can set Pop-ups and redirects to Default, Allow, or Block.
  • Automatic popup blocks are session events, not saved site denies.

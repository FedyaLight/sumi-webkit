Storage Access Frame

Browser UI Expectations

  • This frame page is the requesting embedded origin in a two-origin, two-port setup.
  • document.requestStorageAccess() should trigger Sumi's Storage Access prompt when embedded cross-origin and supported by WebKit.
  • Dismiss and deny outcomes should reject or resolve false according to WebKit behavior without writing unintended allows.

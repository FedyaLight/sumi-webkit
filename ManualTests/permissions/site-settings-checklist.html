Site Settings Checklist

Browser UI Expectations

  • URL hub -> Permissions is current-site scoped and keeps runtime controls separate from saved decisions.
  • Privacy -> Site Settings is profile scoped and lists saved decisions, recent activity, category pages, and site details.
  • Reset permissions clears only permission decisions for the exact scope; cookies and website data remain separate.
  • Unsupported content settings are explanatory only, not fake controls.

Controls

This checklist has no scripted permission requests. Use it while moving through Sumi UI after exercising the per-permission pages.

Results

Manual checklist only.

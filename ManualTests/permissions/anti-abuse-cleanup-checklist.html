Anti-Abuse And Cleanup Checklist

Browser UI Expectations

  • Repeated prompt dismissals enter local cooldown or embargo without writing a saved site Block.
  • Manual Allow, Allow while visiting, or Reset for the exact key bypasses or clears prompt suppression.
  • System-blocked repeated requests are throttled as system UI spam but remain system-blocked, not site-denied.
  • Automatic cleanup is opt-in, removes only stale saved Allow decisions, and records auto-revoked recent activity.

Controls

This checklist coordinates manual steps across the permission pages. Cleanup timing is validated mostly through tests because waiting 90 days manually is not practical.

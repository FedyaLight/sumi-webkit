Geolocation

Browser UI Expectations

  • First getCurrentPosition or watchPosition request shows the URL-bar authorization prompt.
  • URL hub -> Permissions shows Location, saved Ask/Allow/Block state, macOS system status, and Active now controls during provider delivery.
  • Runtime controls support Pause location sharing, Resume location sharing, and Stop sharing location this visit when the provider is active.
  • Location Services disabled or app-denied macOS Location should show system-blocked UI and must not store a site Block.

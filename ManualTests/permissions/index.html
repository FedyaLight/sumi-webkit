Serve this directory from http://localhost or https://localhost. Do not run these pages from file://; secure-context and origin checks are part of the validation.

Browser Runtime

Expected Sumi UI Surfaces

  • URL-bar permission indicator: one current-page icon for pending, active, blocked, system-blocked, or reload-required states.
  • Authorization prompt: URL-bar anchored prompt for camera, microphone, location, notifications, screen sharing, storage access, and user-click external app links.
  • URL hub Permissions submenu: current-site rows, Reset permissions for this site, active runtime controls, and Site settings link.
  • Privacy -> Site Settings: profile-scoped stored decisions, site detail, category pages, reset, cleanup toggle, and recent activity.
  • Runtime controls: camera/microphone mute or stop, geolocation pause/resume/stop-this-visit, and autoplay Reload to apply.
  • Anti-abuse and cleanup: repeated dismissals suppress prompts without stored deny; stale saved Allow cleanup is opt-in and exact-key scoped.
